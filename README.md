# IPL 2024 Performance Dashboard (Power BI)

This repository contains a Power BI dashboard analyzing IPL 2024 using ball-by-ball and innings-level data.
The dashboard focuses on clean metrics, phase-wise performance, and role identification for batters and bowlers.
